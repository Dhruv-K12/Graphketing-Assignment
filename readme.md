# 📱 My React Native App

This is a React Native application built using Expo. The app demonstrates clean architecture, API integration, and a smooth user experience with search and error handling.

---

## 🚀 Features

* 📋 User listing / data display
* 🔍 Search functionality (by name & email)
* ⚡ Fast and responsive UI
* 🌐 API integration using Axios
* ❌ Empty state & error handling

---

## 🛠 Tech Stack

* React Native
* Expo
* TypeScript
* Axios

---

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/Dhruv-K12/Graphketing-Assignment.git
```

2. Install dependencies:

```bash
npm install
```

3. Start the app:

```bash
npx expo start
```

---

## 📁 Folder Structure

```
src/
 ├── components      # Reusable UI components
 ├── screens         # App screens
 ├── api             # API calls (Axios)
 ├── hooks           # Custom hooks
 ├── const           # Constants (colors, fonts, etc.)
 └── navigation      # Navigation setup
```
